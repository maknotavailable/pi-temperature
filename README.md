# pi-temperature
Raspberry Pi Application to read the ambient temperature

## Setup
Install the following:
- rpi_rf
- 433Utils
- WiringPi

## Hardware
- Raspberry Pi Zero W (16GB)
- DS18B20 Temperature Sensor
